  • Progressive Enhancement

    For me, progressive enhancement is like a intruction for web designer to create a website where the core content of a website must be there no matter what. It can be look mordern, super beautiful in this browser, it can be ugly in that browser but the content must always available. The progrssive enhancement is an important in mordern web development because today we have a lot of things we can browse the internet from a smart watch, to a television, or even a refrigerator. So it is important to focus on the core content of the website first, so everyone from every deivces no matter the performance, the browser they are using. Youtube
